# WebFundamentals

## Summary

基礎的なHTML5，CSS3の書き方を学び，レスポンシブWebデザインに対応したスタティックWebページをEmmetを用いて作成する。またCSSフレームワークのBootStrapやJavaScriptとの連携も行う。

## Target

- Emmetを使用して静的なHTMLを作成できる
- 最低限のCSSを使用できる
- HTMLとJavaScriptを連携できる

## Contents

- HTML5
- CSS3
- Emmet
- BootStrap
