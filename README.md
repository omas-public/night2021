# 2021年度 夜間エンジニア科

## Summary

この学科では，UNIX(Linux)を使用してWebおよびプログラミングの基礎を学習し，Reactを用いたWebアプリケーションを作成できるエンジニアを育成する。

またCompTIA ITF+, SEA/J CSBM, AWS Cloud Practitioner を学習し 資格合格を目指す。
